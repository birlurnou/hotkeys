**Скрипт для создания хоткеев**

Все хоткеи полностью настраиваются в файле data.json, который обязательно должен лежать в папке со скриптом  
> Синтаксис json должен строго соблюдаться

С помощью хоткеев можно открыть:

1. Ссылки
	> Список ссылок / одна ссылка  
	> С указанием браузера - в указанном браузере  
	> Без указания браузера - в браузере по умолчанию

2. Программы и файлы
	> Список программ / одна программа без аргументов  
	> Одна программа со списком аргументов / с одним аргументом / без аргументов  
	> Примеры можно посмотреть в файлах data в репозитории

3. Папки

**Какие клавиши можно использовать?**

Сочетание любых клавиш
```json
{  
  "key": [
    "ctrl", "alt", "shift", 
    "cmd", "win", "windows", 
    "esc", "enter", "tab", 
    "space", "backspace", 
    "delete", "insert", 
    "home", "end", "page_up", "page_down", 
    "up", "down", "left", "right",
    "f1 - f12", 
    "Aa - Zz", 
    "0 - 9"
  ]
}

```

**Структура хоткеев**

URL
```json
{  
  "key": "ctrl + alt + 1",  
  "type": "url",  
  "url": [""],  
  "browser": ""  
}
```

PROGRAM
```json
{  
  "key": "ctrl + alt + 2",  
  "type": "program",  
  "path": [""],  
  "arg": [""]  
}
```

FOLDER
```json
{  
  "key": "ctrl + alt + 3",  
  "type": "folder",  
  "path": [""]  
}
```
